# Contributing

Contributions should improve the library as a reusable research source for Remnant Research and Revenants-True-Evolution.

Good contributions include:

- adding verified metadata for papers already discussed in either project
- adding full paper files with clear redistribution permission
- adding topic tags and short relevance notes
- correcting source URLs, titles, authors, or license metadata
- adding import scripts or validation tools

Please do not add full-text papers unless the license allows redistribution.
