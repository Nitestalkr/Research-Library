# Paper and Reference Material Policy

This repository is public. Do not commit copyrighted full-text papers unless redistribution is permitted.

## Allowed

- Papers with explicit open licenses.
- Public-domain papers.
- Author- or publisher-permitted redistribution.
- Project-authored notes, summaries, indexes, and metadata.
- Links, citations, abstracts, tags, and relevance notes.

## Needs Review Before Committing Full Text

- ArXiv papers without an explicit permissive license.
- Publisher PDFs.
- Conference proceedings with unclear redistribution terms.
- Any PDF copied from a private subscription source.

## Metadata Requirements

Every full-text file committed under `papers/` should have a sibling `.json` metadata file with:

- title
- authors
- source URL
- license URL or license statement
- retrieval date
- source project
- topics
- notes on why redistribution is allowed

## Practical Rule

If the license is not clear, do not commit the PDF. Commit the citation, source link, and project notes instead.
